OpenBEL

Language

The Biological Expression Language (BEL) is a language for representing scientific findings in the life sciences in a computable form. BEL is designed to represent scientific findings by capturing causal and correlative relationships in context, where context can include information about the biological and experimental system in which the relationships were observed, the supporting publications cited and the curation process used.

Communication Channels

  • Real-time communication: Gitter

    Gitter is used for real-time chat. You can reference OpenBEL issues like OpenBEL/bel.rb#73. You can also mention one or all users in the chat which will provide them with a desktop notification.

  • BEL Language Questions - how to capture biology in BEL: StackExchange

    StackExchange has the best Q&A system available with the ability to show multiple potential answers and allow users to vote on the best ones.

  • Discussion: Google Groups

    Google Groups is used for OpenBEL discussions and troubleshooting help.

  • Face-to-face Meetings: Google Hangouts

    Google Hangout can be used to hold face-to-face meetings.

    Follow instructions on the Google Hangouts page.
